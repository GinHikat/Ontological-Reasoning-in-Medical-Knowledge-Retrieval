# Experiment Configs

This directory is reserved for future config-driven pipeline experiments.

Current implementation uses code-based builders in `modules/pipelines/factory.py`:

- `legacy_v5`
- `v5_refactored`
- `v6_refined`
- `v7_structured`
- `v8_candidate_integrity`
- `v8_candidate_rescue`

Future versions can add YAML/JSON configs for threshold search, ablation studies, and component swaps.
